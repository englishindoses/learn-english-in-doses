Conditionals

Introduction

Conditionals are sentence structures used to discuss possibilities, hypothetical situations, and their consequences. They help us talk about real and unreal situations in the present, future, and past. English has several types of conditionals, each used for different situations with varying degrees of probability.

After this lesson, you will be able to:

  • Identify and use the five conditional types correctly (zero, first, second, third, and mixed).
  • Express real, possible, and hypothetical situations with the appropriate conditional form.
  • Create more nuanced expressions using mixed conditionals to connect past and present.

Zero and First Conditionals (Real conditions)

Zero Conditional

Form: If + present simple, present simple

When to use it: Use the zero conditional for general truths, facts, and things that always happen as a result of something else. There is a 100% chance that the condition leads to the result.

  1. If you heat water to 100¬∞C, it boils. (Scientific fact)
  2. If I don't drink coffee in the morning, I get a headache. (Personal fact)
  3. If you press this button, the machine starts. (Instruction/general truth)

First Conditional

Form: If + present simple, will + infinitive

When to use it: Use the first conditional for real possibilities in the future - things that are likely to happen if a certain condition is met. There is a strong possibility that the condition will be fulfilled.

  1. If it rains tomorrow, I will stay at home. (Possible future situation)
  2. If you finish this report today, the boss will be very pleased. (Likely outcome at work)
  3. If we don't hurry, we will miss our flight. (Warning about future consequence)

Native speaker note: In conversation, native speakers often use "going to" or present continuous instead of "will" in the first conditional: "If it rains tomorrow, I'm going to stay home" or "If you finish this report, the boss is giving you a bonus."

Second and Third Conditionals (Unreal conditions)

Second Conditional

Form: If + past simple, would + infinitive

When to use it: Use the second conditional for unreal or unlikely situations in the present or future. These are hypothetical situations or things that are improbable.

  1. If I won the lottery, I would buy a house. (Unlikely, but possible in future)
  2. If I spoke Chinese, I would apply for that job. (I don't speak Chinese now)
  3. What would you do if you had more free time? (Hypothetical present situation)

Third Conditional

Form: If + past perfect, would have + past participle

When to use it: Use the third conditional for impossible situations in the past - things that didn't happen, but we imagine what the result would have been if they had happened.

  1. If I had studied harder, I would have passed the exam. (I didn't study hard enough, so I failed)
  2. If she had arrived earlier, she would have met the CEO. (She didn't arrive early, so she didn't meet the CEO)
  3. If they hadn't missed their train, they would have been on time. (They missed their train, so they weren't on time)

Native speaker note: Native speakers often use contractions in conditionals: "If I'd studied harder, I would've passed" (third conditional) or "If I had more time, I'd learn Spanish" (second conditional).

Mixed Conditionals

Form: There are two common mixed conditional patterns:

1. If + past perfect, would + infinitive (Past condition ‚Üí Present result)

2. If + past simple, would have + past participle (Present condition ‚Üí Past result)

When to use them: Use mixed conditionals when the time in the if-clause is different from the time in the main clause. They're used to connect past and present situations.

Type 1: Past condition ‚Üí Present result

  1. If I had studied medicine, I would be a doctor now. (I didn't study medicine in the past, so I'm not a doctor now)
  2. If she hadn't moved to London, she wouldn't work for that company now. (She moved to London in the past, so she works for that company now)
  3. If he had taken that job in Paris, he would live abroad now. (He didn't take the job, so he doesn't live abroad now)

Type 2: Present condition ‚Üí Past result

  1. If I weren't afraid of flying, I would have visited Japan last year. (I am afraid of flying now, so I didn't visit Japan)
  2. If she spoke French, she would have applied for that position. (She doesn't speak French now, so she didn't apply)
  3. If I liked coffee, I would have ordered an espresso earlier. (I don't like coffee now, so I didn't order one)

Native speaker note: Mixed conditionals can be difficult even for native speakers. They're used more in written English or formal speech rather than casual conversation. In everyday conversation, people might use simpler forms to express these complex ideas.

Common Errors

1: Using will in the if-clause

If it will rain tomorrow, I will stay at home.


If it rains tomorrow, I will stay at home. ‚úì

We don't use "will" in the if-clause of first conditionals. Use present simple instead.

2: Mixing conditional types incorrectly

If I would have money, I will buy a new car.


If I had money, I would buy a new car. ‚úì

This is a second conditional (unlikely present/future situation). Use past simple in the if-clause and would + infinitive in the main clause.

3: Using was instead of were in second conditionals

If I was rich, I would travel the world.


If I were rich, I would travel the world. ‚úì

In formal English, we use "were" for all persons (I, he, she, it) in second conditional if-clauses. However, in everyday English, "was" is increasingly accepted.

Useful Tips

üìù TIP 1: Think of conditionals on a probability scale: Zero (100% - facts), First (likely future), Second (unlikely present/future), Third (impossible past). This helps you choose the right form based on how probable the situation is.

üìù TIP 2: Remember the fixed verb tense patterns for each conditional type and learn them as formulas. Create a flashcard for each type with the formula and a simple example you can easily remember.

üìù TIP 3: Practice constructing mixed conditionals by first identifying whether the condition is in the past or present, and whether the result is in the past or present. This will help you choose the correct structure.
