Multiple Choice Questions

1. I'll lend you the book _____ you return it to me next week.

A. unless
B. although
C. provided that
D. despite

2. _____ it rains, we'll have the picnic inside.

A. Unless
B. If
C. Provided that
D. As long as

3. You can use my laptop _____ you don't download anything.

A. provided that
B. if not
C. as long as
D. unless
