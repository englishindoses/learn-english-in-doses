There Is / There Are

Introduction

We use "there is" and "there are" to say that something exists or to describe what is in a place. You use this structure every day — for example, to describe a room, a city, or a menu. "There is" is for one thing (singular). "There are" is for more than one thing (plural).

After this lesson, you will be able to:

  • Use "there is" and "there are" in positive, negative, and question sentences.
  • Choose between "there is" (singular/uncountable) and "there are" (plural).
  • Combine "there is/are" with "some", "any", and "a/an".

1. Positive Sentences

Form: there is + singular / uncountable noun. There are + plural noun.

When to use it: Use "there is/are" to say that something exists in a place.

There is (singular / uncountable):

  1. There is a park near my house.
    (= A park exists near my house.)
  2. There is a good restaurant on this street.
    (= A restaurant exists on this street.)
  3. There is some milk in the fridge.
    (= Milk exists in the fridge. Milk is uncountable.)

There are (plural):

  1. There are three bedrooms in the flat.
    (= Three bedrooms exist.)
  2. There are some chairs in the meeting room.
    (= Chairs exist in the room.)
  3. There are many shops at the airport.
    (= Shops exist at the airport.)

Short forms: In speaking, we often say "There's" instead of "There is". "There's a café on the corner." There is no short form for "there are".

2. Negative Sentences

Form: there isn't + singular / uncountable noun. There aren't + plural noun.

When to use it: Use the negative to say that something does NOT exist in a place.

  1. There isn't a swimming pool at the hotel.
    (= No swimming pool exists.)
  2. There isn't any coffee left.
    (= Zero coffee. Coffee is uncountable.)
  3. There aren't any shops near here.
    (= Zero shops.)

Remember: Use "any" in negative sentences. "There isn't any bread." "There aren't any eggs."

3. Questions

Form: is there + singular / uncountable noun + ? Are there + plural noun + ?

When to use it: Swap "there" and "is/are" to make questions.

  1. Is there a bathroom on this floor?
    (= Does a bathroom exist here?)
  2. Is there any water?
    (= Does water exist?)
  3. Are there any good restaurants near the hotel?
    (= Do restaurants exist near the hotel?)

Short answers:

  1. Is there a lift? — Yes, there is. / No, there isn't.
  2. Are there any seats? — Yes, there are. / No, there aren't.

Remember: Use "any" in questions. "Is there any milk?" "Are there any tickets?"

4. "How Many" + "There Are"

Form: how many + plural noun + are there + ?

When to use it: Use "how many... are there?" to ask about the number of things in a place.

  1. How many rooms are there in the hotel?
    (= I want to know the number of rooms.)
  2. How many people are there in your team?
    (= I want to know the number of people.)
  3. How many stops are there before the airport?
    (= I want to know the number of stops.)

Summary: There Is / There Are

Type Singular / Uncountable Plural
Positive There is a book. / There is some water. There are some books.
Negative There isn't a book. / There isn't any water. There aren't any books.
Question Is there a book? / Is there any water? Are there any books?
Short answer (+) Yes, there is. Yes, there are.
Short answer (−) No, there isn't. No, there aren't.

Common Errors

Using "there is" with plural nouns

There is many people in the room.

There are many people in the room.

"People" is plural, so use "there are".

Confusing "there" and "their"

Their is a café on the corner.

There is a café on the corner.

"There is/are" talks about existence. "Their" is a possessive (their car, their house).

Using "have" instead of "there is/are"

The hotel has a swimming pool. (when describing what exists)

There is a swimming pool at the hotel.

When you describe what exists in a place, use "there is/are". "The hotel has..." is not wrong, but "there is..." is more natural when describing a place to someone.

How to Remember and Practice

📝 Tip: Look around the room you are in right now. Describe it: "There is a window. There are two chairs. There is a lamp on the desk." This is a quick and easy way to practise.

📝 Tip: The simple rule: ONE thing = "there is". TWO or MORE things = "there are". If you can count more than one, use "are".

📝 Tip: When you travel, describe your hotel room or the area around you: "There is a bed. There are two pillows. There isn't a minibar. Are there any towels?" This is real-life practice.
