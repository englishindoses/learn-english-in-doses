Appendix – Random Questions Generator (HTML)

Instructions for Download

This pack doesn’t go into detail about HTML files, and the full file would be too long to include here (it contains 100 questions). This appendix explains how to download and open the interactive HTML file so that you can use the Random Questions Generator you created in your class.

1 – Download the File

Click on the drop-down arrow next to Copy for more options. Then click on “Download as HTML”.

2 – Open the File

When the download is finished, click on Downloads and then click on the file. The file will open in your browser.
